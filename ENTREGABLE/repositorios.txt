# Repositorios del Proyecto SmartLogix

## Repositorio Principal
https://github.com/JONAHBRUZZI/smartlogix

Contiene todo el proyecto SmartLogix:
- Frontend/ — Aplicacion SPA en React + Vite + TypeScript (sistema interno POS, inventario, pedidos, despachos)
- Landing/ — Landing page en Next.js 16 + Tailwind CSS v4
- Backend/ — 4 microservicios Node.js/Express + API Gateway Nginx + PostgreSQL
- ENTREGABLE/ — Documentacion (analisis de patrones, plan de branching)

## Microservicios Backend

### orders-service
- Codigo: Backend/orders-service/
- Descripcion: Gestion de pedidos. CRUD de ordenes, cambio de estados, confirmacion y cancelacion. Base de datos: orders_db.
- Patrones: Repository (pg), Observer (SQS)
- Tecnologias: Node.js 22, Express, pg, AWS SDK SQS
- Puerto: 8081

### inventory-service
- Codigo: Backend/inventory-service/
- Descripcion: Control de inventario. Gestion de productos, ajustes de stock, ventas. Base de datos: inventory_db.
- Patrones: Repository (pg), Observer (SQS)
- Tecnologias: Node.js 22, Express, pg, AWS SDK SQS
- Puerto: 8082

### shipping-service
- Codigo: Backend/shipping-service/
- Descripcion: Despachos. Generacion de envios desde pedidos confirmados, seguimiento con codigo QR, confirmacion de entrega. Base de datos: shipping_db.
- Patrones: Repository (pg), Observer (SQS), REST notifications
- Tecnologias: Node.js 22, Express, pg, AWS SDK SQS
- Puerto: 8084

### notification-service
- Codigo: Backend/notification-service/
- Descripcion: Notificaciones. Persistencia y consulta de eventos de trazabilidad via REST. Base de datos: notification_db.
- Patrones: Repository (pg)
- Tecnologias: Node.js 22, Express, pg
- Puerto: 8085

## Frontend

### Sistema Interno (SPA)
- Codigo: Frontend/
- Descripcion: Aplicacion React SPA con Vite + TypeScript. Modulos: Dashboard, POS, Inventario, Pedidos, Despachos, Clientes, Reportes, Usuarios.
- Patrones: Custom Hook (useApiQuery, useOperationalWorkspace), Adapter (api-adapters), Proxy (api-client), Composite (useOperationalWorkspace)
- Tecnologias: React 19, Vite 6, TypeScript, Tailwind CSS

### Landing Page
- Codigo: Landing/
- Descripcion: Landing page de SmartLogix. Secciones de Hero, Caracteristicas, Como Funciona, Planes, Demo y FAQ.
- Patrones: Componentes atomicos
- Tecnologias: Next.js 16, React 19, Tailwind CSS v4
- URL: https://smartlogix-landing.vercel.app

## Infraestructura

### Docker Compose
- node.yml: Desarrollo con build local
- vm.yml: VM con imagenes pre-built (Docker Hub)
- optimized.yml: Produccion optimizado (1GB RAM)
- prod.yml: Produccion completo con elasticmq

### Base de Datos
- Motor: PostgreSQL 15 (Alpine)
- Bases: orders_db, inventory_db, shipping_db, notification_db
- Scripts: Backend/init-db.sql (creacion), Backend/seed.sql (datos de prueba)

### Mensajeria
- Dev: ElasticMQ (imagen softwaremill/elasticmq)
- Prod: AWS SQS
- Colas: orders-queue, shipping-queue
- Config: elasticmq.conf
